Interface ISubscriptionObserver

Describes an observer interface which should be implemented in order to listen for subscription events. Use an instance of the SubscriptionObserver class to provide the IRtmClient.CreateSubscription(String, SubscriptionModes, ISubscriptionObserver) method with a subscription observer.

Parameters
Type Name Description
ISubscription subscription
Remarks

The subscription enters a subscribed state when a subscribe request to the RTM services succeed. During this state the subscription can receive data, info and error. See the OnSubscriptionData(ISubscription, RtmSubscriptionData), OnSubscriptionInfo(ISubscription, RtmSubscriptionInfo), and OnSubscriptionError(ISubscription, RtmSubscriptionError) methods.
